Código de Conducta

tl;dr: Be nice. Be responsible.

Code Mazatlán intenta a ser un lugar seguro, libre de acoso para nuestros invitados, sin importar raza, etnia, edad, género, orientación sexual, religión, discapacidad, aparencia física, clase social, o creencias politicas.

Además, hay que respetar y seguir las polizas de UBICACIÓN.

Y no manejes bajo la influencia de alcohol!
